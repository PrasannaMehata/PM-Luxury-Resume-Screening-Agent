# PM-Luxury-Resume-Screening-Agent
A beautiful, animated, premium-design AI resume screening system built with NLP, embeddings, and vector search.

Features :
- Resume Processing

- Upload multiple resumes

- Supports PDF, DOCX & TXT

- Extracts text using pdfplumber & PyPDF2

- AI-Powered Analysis

- TF-IDF text similarity

- OpenAI embeddings similarity

- ChromaDB vector search

- Skill extraction

- Experience estimation

- Education detection

- Contact extraction

- Auto-summary

# Premium UI

- Gradient animated header

- Luxury cards

- Progress bars

- Top performers view

- Candidate comparison view

# Exports

- All candidates CSV

- Recommended candidates only

- Top performers CSV
